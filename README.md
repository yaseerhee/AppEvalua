# SISTEMA DE GESTIÓN DE NOTAS :mortar_board:

## DESCRIPCIÓN :page_with_curl:
Se trata de una aplicación que sige los criterios de evluación del profesor de JavaScript y calcula las notas de los alumnos.

## EVALUACIÓN :100:
- [X] **Tratamiento de casos raros** -> 10% 
- [X] **Funcionamiento** -> 70% 
- [X] **Calidad, comentarios y nomenclatura** -> 15% 
- [X] **Diseño** -> 5%
